HTIRSeeker PLUS

The HTIRSeeker PLUS is an extended programming block for LEGO MINDSTORMS EV3. In addition to measuring “direction” and “strength (intensity)” like the original IRSeeker, this block also provides comparison and selection modes that can be used with Switch and Loop blocks.

For more details, please visit the GitHub repository.

Modes

Measure - AC/DC Direction

Measures the direction of the infrared signal. It can detect the IR ball signal direction used in RoboCup Junior Soccer (A-mode / B-mode).

Measure - AC/DC Strength

Measures the strength (intensity) of the infrared signal. Useful for estimating the distance to the IR ball.

Select - AC/DC Direction

A direction selection mode for use with the Switch block. You can access this mode by selecting “HTIRSeekerPLUS” → “Measure” → “AC/DC Mode” → “Direction”.

Compare - AC/DC Strength

Compares the detected infrared strength with a user-defined threshold using a comparison operator. The result is returned as TRUE or FALSE, and can be used in Switch, Loop, and Wait blocks.
